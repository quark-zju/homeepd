Code I use for e-paper displays at home.

- dev-133e6: [Waveshare 13.3inch e-Paper](<https://www.waveshare.com/wiki/13.3inch_e-Paper_HAT+_(E)>). Custom Raspberry Pi + [PiSugar 3](https://github.com/PiSugar/PiSugar/wiki/PiSugar-3-Series) logic to download and refresh images daily.
- dev-7e6: [Waveshare 7inch PhotoPainter](https://www.waveshare.com/wiki/ESP32-S3-PhotoPainter). Modified stock ESP32 code to download and refresh images daily.
